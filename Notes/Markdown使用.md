# 2.斜体、加粗、删除线、分割线

在文本两端加上1个星号为*斜体*

文本两端加上2个星号为**加粗**或者加上2个英文的下划线__加粗__

如果又想斜体又想加粗则在文本两端加上3个星号***斜体加粗***

在文本两端加上两个波浪号为添加 ~~删除线~~

如果想添加一条分割线只用在两端文本中间添加3个星号

# 4.代码块

在代码的开始和结束行各加入**3个~**，如果要区分语言则在开始的3个波浪键后加入对应语言名称

这是C语言

~~~C
#include <stdio.h>
int main(){
	printf("Hello, World");
    return(0);
}
~~~

这是Java语言

~~~Java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
~~~

**如果想要在文字中加入代码块则在内容前后加入`**

这是C语言中的`include`头文件

# 6.超链接

把想要跳转的文本用[ ]括起来后面加上()括号里面输入网址

> 调换到[百度](www.baidu.com)网址  

如果要添加多个网址可以用占位符表达，在[]后面加个[a]，然后只要在下面补充就行

>去[百度][a]，去[必应][b]
>[a]:www.baidu.com
>[b]:www.bing.com

如果要添加注释只用在[]中添加一个^即可

> 这是注释[^1]
>
> [^1]:我是脚注

# 9.数学公式

单行的数学公式首尾用$包围

$y=1+x$

多行的数学公式用2个$包围。**两个公式要换行中间用2个\隔开**

$$
1+x=y\\
2+y=x
$$

***

常用格式：

1. $\frac{1}{2}$

2. $$
	x^2\\
	x_2
	$$

3. $\sqrt[3]{4}$  方括号中可以省略

***
